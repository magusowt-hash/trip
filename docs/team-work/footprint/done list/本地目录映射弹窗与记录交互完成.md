# 本地目录映射弹窗与记录交互完成

## 改动范围
- `src/components/LocalMapModal.tsx`
- `src/components/LocalMapModal.module.css`
- `src/app/(shell)/user/footprints/page.tsx`

## 改动目标
将本地目录映射第一阶段弹窗从草案收敛为正式交互，并把主文件夹记录、保存记录、差异提示、未匹配目录与缺失文件警告整合进正式入口。

## 改动思路
以“主文件夹名称为唯一记录键”的第一阶段约束为前提，把原测试与草案里的目录扫描、记录恢复、差异提示能力压缩到一个正式弹窗中，优先保证用户能完成重新选择主文件夹、恢复图片、继续保存这条主链路。

## 改动内容
- 正式接入 `映射本地` 弹窗，并支持：
  - 扫描主文件夹第一层子目录
  - 读取并恢复同名主文件夹的旧记录
  - 展示主文件夹记录 / 保存记录
  - 展示新增文件、变化文件、缺失文件
  - 展示未匹配目录列表与已匹配地点统计
- 优化弹窗结构与文案：
  - 选择主文件夹按钮下移到底部
  - 记录区块独立为标题下方卡片
  - “已知主文件夹 / 旧记录”文案改为“主文件夹记录 / 保存记录”
- 完成导入后：
  - 已匹配图片进入足迹页
  - 缺失文件在确认保存后删除原位置记录
  - 未匹配目录不进入正式足迹页
