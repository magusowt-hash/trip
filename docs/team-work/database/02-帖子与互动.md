# 帖子与互动

## posts 帖子表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| user_id | INT | NO | - | 发帖用户 ID（关联 users.id） |
| title | VARCHAR(120) | NO | - | 帖子标题 |
| content | TEXT | YES | NULL | 帖子正文内容 |
| cover_image_url | TEXT | YES | NULL | 封面图 URL |
| privacy | VARCHAR(16) | YES | 'public' | 可见性：public, private, friends |
| topic | VARCHAR(64) | YES | NULL | 话题标签 |
| comments_cnt | INT | YES | 0 | 评论数 |
| favorites_cnt | INT | YES | 0 | 收藏/点赞数 |
| status | VARCHAR(16) | YES | 'normal' | 状态：normal, deleted 等 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | NO | NOW() | 更新时间 |

**索引：**
- 无显式索引（可通过 user_id 查询）

**说明：** 用户发布的帖子主表，包含正文、封面、话题等。

---

## post_images 帖子图片表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| post_id | INT | NO | - | 关联帖子 ID（关联 posts.id） |
| url | TEXT | NO | - | 图片 URL |
| thumbnail_url | TEXT | YES | NULL | 缩略图 URL |
| caption | TEXT | YES | NULL | 图片说明文字 |
| sort_order | INT | YES | 0 | 排序顺序 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |

**索引：**
- 无显式索引

**说明：** 帖子的多图附件，一个帖子可有多张图片。

---

## comments 评论表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| post_id | INT | NO | - | 关联帖子 ID（关联 posts.id） |
| user_id | INT | NO | - | 评论用户 ID（关联 users.id） |
| content | TEXT | NO | - | 评论内容 |
| parent_id | INT | YES | NULL | 父评论 ID（用于回复嵌套） |
| status | VARCHAR(16) | YES | 'normal' | 状态：normal, deleted 等 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |

**索引：**
- 无显式索引

**说明：** 帖子的评论，支持嵌套回复（通过 parent_id 关联）。

---

## favorites 收藏/点赞表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| post_id | INT | NO | - | 关联帖子 ID（关联 posts.id） |
| user_id | INT | NO | - | 收藏用户 ID（关联 users.id） |
| created_at | TIMESTAMP | NO | NOW() | 收藏时间 |

**索引：**
- `favorites_post_user_unique`（UNIQUE）：post_id, user_id（防止重复收藏）

**说明：** 用户对帖子的收藏/点赞记录，每个用户对同一帖子只能收藏一次。

---

## comment_likes 评论点赞表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| comment_id | INT | NO | - | 关联评论 ID（关联 comments.id） |
| user_id | INT | NO | - | 点赞用户 ID（关联 users.id） |
| created_at | TIMESTAMP | NO | NOW() | 点赞时间 |

**索引：**
- `comment_likes_comment_user_unique`（UNIQUE）：comment_id, user_id

**说明：** 用户对评论的点赞记录。

---

## ratings 评分表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| user_id | INT | NO | - | 评分用户 ID（关联 users.id） |
| target_type | VARCHAR(32) | NO | - | 评分对象类型：如 post, list_item, marker |
| target_id | INT | NO | - | 评分对象 ID |
| rating | TINYINT | NO | - | 评分值（1-5） |
| comment | TEXT | YES | NULL | 评分附带的文字评价 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | NO | NOW() | 更新时间 |

**索引：**
- `ratings_user_target_unique`（UNIQUE）：user_id, target_type, target_id（防止重复评分）

**说明：** 用户对各类对象（帖子、榜单项、地点等）的评分记录。
