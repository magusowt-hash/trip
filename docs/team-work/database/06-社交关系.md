# 社交关系

## friendships 好友关系表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| user_id | INT | NO | - | 用户 ID（关联 users.id） |
| friend_user_id | INT | NO | - | 好友用户 ID（关联 users.id） |
| status | VARCHAR(16) | YES | 'pending' | 关系状态：pending, accepted, rejected |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |

**索引：**
- 无显式索引

**说明：** 用户之间好友关系，通过 status 字段区分申请中/已接受/已拒绝状态。
