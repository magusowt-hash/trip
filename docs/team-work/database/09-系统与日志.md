# 系统与日志

## embed_access_logs 嵌入访问日志表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| ip | VARCHAR(45) | NO | - | 访问者 IP 地址 |
| action | VARCHAR(32) | NO | - | 操作类型：view, click, share 等 |
| list_id | INT | YES | NULL | 关联榜单 ID（关联 lists.id） |
| item_id | INT | YES | NULL | 关联榜单项 ID（关联 list_items.id） |
| user_agent | TEXT | YES | NULL | 用户代理字符串 |
| created_at | TIMESTAMP | NO | NOW() | 访问时间 |

**索引：**
- 无显式索引

**说明：** 记录榜单嵌入页面的访问和互动日志，用于统计分析。
