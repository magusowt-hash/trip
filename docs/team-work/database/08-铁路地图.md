## station_overrides 站点覆盖配置表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| station_name | VARCHAR(64) | NO | - | 原始站点名称 |
| display_name | VARCHAR(64) | YES | NULL | 显示名称（覆盖默认） |
| level_override | ENUM | YES | NULL | 级别覆盖：CH(枢纽), RK(主要), GI(地方主要), AS(地方), MT(地铁), deleted |
| display_level | ENUM | YES | NULL | 显示级别：CH, RK, GI, AS, MT |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | NO | NOW() | 更新时间 |

**索引：**
- `station_overrides_name_idx`（UNIQUE）：station_name

**说明：** 对特定站点进行名称或级别的覆盖配置，用于处理特殊站点或数据错误。
