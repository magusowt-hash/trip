# 用户与认证

## users 用户表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| phone | VARCHAR(32) | NO | - | 手机号（唯一登录标识） |
| password_hash | VARCHAR(255) | NO | - | 密码哈希值 |
| nickname | VARCHAR(64) | YES | NULL | 昵称 |
| avatar | TEXT | YES | NULL | 头像 URL |
| gender | TINYINT | YES | 0 | 性别：0=未知, 1=男, 2=女 |
| birthday | DATE | YES | NULL | 生日 |
| region | VARCHAR(128) | YES | NULL | 地区 |
| favorite_lists | JSON | YES | NULL | 历史收藏榜单列表兼容字段，榜单地点收藏主来源已迁至 `user_list_favorites` |
| ratings | JSON | YES | NULL | 用户评分记录（JSON） |
| status | VARCHAR(16) | YES | 'normal' | 状态：normal, banned 等 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | NO | NOW() | 更新时间 |

**索引：**
- 主键：`id`

**说明：** 核心用户表，存储用户基本信息。

---

## user_list_favorites 用户榜单项收藏表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| user_id | INT | NO | - | 用户 ID（关联 users.id） |
| list_item_id | INT | NO | - | 榜单项 ID（关联 list_items.id） |
| created_at | TIMESTAMP | NO | NOW() | 收藏时间 |

**索引：**
- `user_list_favorites_user_item_unique`（UNIQUE）：user_id, list_item_id
- `user_list_favorites_user_id_idx`：user_id
- `user_list_favorites_list_item_id_idx`：list_item_id

**说明：** 榜单页、地点收藏页使用的地点收藏主表；替代 `users.favorite_lists` 作为业务读写来源。

---

## admin_keys 后台管理员密钥表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| key_hash | VARCHAR(255) | NO | - | 密钥哈希值 |
| name | VARCHAR(64) | NO | - | 密钥名称/描述 |
| is_master | TINYINT | YES | 0 | 是否为主密钥（1=是） |
| is_active | TINYINT | YES | 1 | 是否激活（1=激活，0=禁用） |
| expires_at | TIMESTAMP | YES | NULL | 过期时间 |
| last_used_at | TIMESTAMP | YES | NULL | 最后使用时间 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |

**索引：**
- `admin_keys_key_hash_idx`（UNIQUE）：key_hash

**说明：** 后台管理认证密钥，用于管理后台接口的身份验证。

---

## user_footprint_settings 用户足迹设置表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| user_id | INT | NO | - | 主键，对应 users.id |
| show_photos | TINYINT | NO | 1 | 是否显示照片 |
| show_lines | TINYINT | NO | 1 | 是否显示连线 |
| show_labels | TINYINT | NO | 1 | 是否显示标签 |
| show_poi_labels | TINYINT | NO | 1 | 是否显示 POI 标签 |
| poi_label_color | VARCHAR(16) | NO | '#000000' | POI 标签颜色 |
| marker_color | VARCHAR(16) | NO | '#ef4444' | 标记点颜色 |
| marker_shape | VARCHAR(16) | NO | 'pin' | 标记形状 |
| show_title | TINYINT | NO | 1 | 是否显示标题 |
| panel_collapsed | TINYINT | NO | 0 | 面板是否折叠 |
| background_color | VARCHAR(16) | NO | '#0f172a' | 背景颜色 |
| line_color | VARCHAR(16) | NO | '#a5b4fc' | 连线颜色 |
| line_width | DOUBLE | NO | 2 | 连线宽度 |
| line_dashed | TINYINT | NO | 1 | 是否虚线 |

**索引：**
- 主键：user_id

**说明：** 存储用户在足迹地图页面的个性化显示设置。
