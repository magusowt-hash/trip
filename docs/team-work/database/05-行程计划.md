# 行程计划

## plans 计划表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| user_id | INT | NO | - | 用户 ID（关联 users.id） |
| name | VARCHAR(120) | NO | - | 计划名称 |
| status | VARCHAR(16) | YES | 'active' | 状态：active, completed, cancelled |
| active_tab | INT | YES | 0 | 当前活跃的 Tab 页索引 |
| start_date | DATE | YES | NULL | 计划开始日期 |
| end_date | DATE | YES | NULL | 计划结束日期 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | NO | NOW() | 更新时间 |

**索引：**
- `plans_user_id_idx`（UNIQUE）：user_id

**说明：** 用户行程计划的主表，记录行程名称、时间范围和活跃状态。

## transport_items 交通项表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| plan_id | INT | NO | - | 关联计划 ID（关联 plans.id） |
| from | VARCHAR(255) | YES | NULL | 出发地/起点 |
| to | VARCHAR(255) | YES | NULL | 目的地/终点 |
| note | TEXT | YES | NULL | 备注说明 |
| note_expanded | TINYINT | YES | 0 | 备注是否展开（1=展开） |
| sort_order | INT | YES | 0 | 排序顺序 |
| start_date | DATE | YES | NULL | 交通开始日期 |
| end_date | DATE | YES | NULL | 交通结束日期 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |

**索引：**
- `transport_items_plan_id_idx`（UNIQUE）：plan_id, sort_order

**说明：** 行程计划中的交通信息（飞机、火车、自驾等），from/to 可存出发地和目的地。
