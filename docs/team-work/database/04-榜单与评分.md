# 榜单与评分

## lists 榜单表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| name | VARCHAR(255) | NO | - | 榜单名称 |
| cover_image | TEXT | YES | NULL | 封面图 URL |
| description | TEXT | YES | NULL | 描述 |
| position | INT | YES | NULL | 在榜单页中的排序位置 |
| intro | TEXT | YES | NULL | 榜单简介 |
| lng | VARCHAR(20) | YES | NULL | 关联位置经度 |
| lat | VARCHAR(20) | YES | NULL | 关联位置纬度 |
| status | TINYINT | YES | 1 | 状态：1=启用，0=禁用 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | NO | NOW() | 更新时间 |

**索引：**
- 无显式索引

**说明：** 用户或系统创建的地点榜单（如"成都美食榜单"），可关联地理位置。

---

## list_images 榜单图片表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| list_id | INT | NO | - | 关联榜单 ID（关联 lists.id） |
| url | TEXT | NO | - | 图片 URL |
| thumbnail_url | TEXT | YES | NULL | 缩略图 URL |
| caption | TEXT | YES | NULL | 图片说明 |
| sort_order | INT | YES | 0 | 排序顺序 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |

**索引：**
- `list_images_list_id_idx`：list_id

**说明：** 榜单的多张展示图片。

---

## list_items 榜单项表

| 列名 | 类型 | 可空 | 默认值 | 说明 |
|------|------|------|--------|------|
| id | SERIAL | NO | 自增 | 主键 |
| list_id | INT | NO | - | 关联榜单 ID（关联 lists.id） |
| title | VARCHAR(255) | NO | - | 地点/项标题 |
| cover_image | TEXT | YES | NULL | 封面图 URL |
| description | TEXT | YES | NULL | 描述 |
| intro | TEXT | YES | NULL | 简介 |
| image_url | TEXT | YES | NULL | 图片 URL |
| lng | VARCHAR(20) | YES | NULL | 经度 |
| lat | VARCHAR(20) | YES | NULL | 纬度 |
| address | VARCHAR(500) | YES | NULL | 地址 |
| order_num | INT | YES | 0 | 排序序号 |
| status | TINYINT | YES | 1 | 状态：1=启用，0=禁用 |
| created_at | TIMESTAMP | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | NO | NOW() | 更新时间 |

**索引：**
- 无显式索引

**说明：** 榜单中的具体地点项，包含位置坐标和图文信息。
